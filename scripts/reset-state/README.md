# reset-state

Clear the background color on all selected elements in one step.

## How to use

1. Select the elements you want to reset (or select all with Ctrl/Cmd+A).
2. Run **reset-state** from the Tools Panel.

All selected shapes have their background set to `transparent`. Text elements, arrows, and lines are unaffected.

## Notes

- This is effectively a shortcut for undoing all [toggle-state](../toggle-state/README.md) highlights at once.
- Useful at the start of a new algorithm trace on an existing diagram.
